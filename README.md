# MED_Desktop - Десктопная CRM для больницы

Нативное десктопное приложение для управления больничной CRM-системой.

## Установка

```bash
# Создать виртуальное окружение
python -m venv venv

# Активировать
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac

# Установить зависимости
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Тестовые учётные данные

```
# Регистратор
Логин: reg1
Пароль: reg123

# Врач (кардиология)
Логин: doc1
Пароль: doc123

# Начальник отделения (кардиология)
Логин: lead1
Пароль: lead123

# Медсестра (кардиология)
Логин: nur1
Пароль: nur123

# Администратор
Логин: admin
Пароль: admin123
```

## Роли и права доступа

| Роль | Код | Права |
|------|-----|-------|
| Администратор | ADMIN | Полный доступ ко всем функциям |
| Регистратор | REG | Создание/редактирование пациентов, скрытие пациентов, создание пользователей |
| Начальник отделения | LEAD | Доступ к пациентам своего отделения, управления планом лечения |
| Врач | DOC | Доступ только к своим пациентам, визиты, заметки, назначения |
| Медсестра | NUR | Доступ к пациентам своего отделения, визиты, заметки |
