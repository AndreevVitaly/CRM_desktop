## 3. ИСПРАВЛЕНИЕ МОДЕЛЕЙ ДАННЫХ
================================================================================

### 3.1 Добавлено поле middle_name в User
Файл: models/db_models.py
Изменения:
  - Добавлено поле middle_name: str = "" в dataclass User
  - Обновлён метод save() для сохранения middle_name
  - Обновлён метод full_name для отображения ФИО полностью
  - Добавлено поле в SQL-запрос создания таблицы users

### 3.2 Конвертация дат из БД
Файл: models/db_models.py
Проблема: Даты из SQLite приходили как строки, вызывая AttributeError
Решение: Добавлен метод _from_row() для всех моделей с датами:
  - Patient: birth_date, created_at
  - Encounter: started_at, finished_at
  - Note: created_at
  - TreatmentPlanItem: due_date, created_at, updated_at
  - Event: event_date, event_time, created_at
  
Поддерживаемые форматы дат:
  - %Y-%m-%d %H:%M:%S.%f
  - %Y-%m-%d %H:%M:%S
  - %Y-%m-%dT%H:%M:%S.%f
  - %Y-%m-%dT%H:%M:%S
